# Downloadable Datasets Guide

This guide explains how to access and download public datasets from RegulonDB.

## Entry Point

- Main datasets page: [https://regulondb.ccg.unam.mx/datasets](https://regulondb.ccg.unam.mx/datasets)

## What You Can Download

The datasets page provides grouped resources such as:

- Regulatory interactions (`RISet`, `TF-RISet`)
- Regulators (`RegulatorSet`, `TFSet`)
- Regulatory network datasets
- Gene-related datasets (`GeneProductSet`, sequence and identifier sets)
- Operon/transcription-structure datasets (`OperonSet`, `TUSet`, `PromoterSet`, `TerminatorSet`)
- Evidence catalogs (`EvidenceSet`, `AdditiveEvidenceSet`)

## Download Workflow

1. Open the datasets page.
2. Locate the dataset category and file.
3. Use the **Download** option to export data.
4. Optionally use the **Browse/Filter** view for selective extraction.
5. Optionally use the confidence tool when supported by the dataset.

## Recommended Usage

- Use full downloads for reproducible local analysis.
- Use filtered exports when building focused subsets.
- Record dataset name, date, and source URL for traceability.
