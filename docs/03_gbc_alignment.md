# Alineacion con Global Biodata Coalition (GBC)

## Objetivo
Construir una matriz de alineacion entre la documentacion del repositorio y los criterios GBC que el equipo este usando como referencia.

## Ejes propuestos para la matriz
Estos ejes deben confirmarse con el equipo para asegurar que reflejen los criterios GBC vigentes.
- Gobernanza y sostenibilidad
- Acceso abierto y licencias
- Calidad, curacion y trazabilidad
- Interoperabilidad y estandares
- Preservacion y continuidad del servicio
- Comunidad, soporte y capacitacion
- Etica, privacidad y seguridad

## Matriz sugerida
- Criterio GBC
- Evidencia documental
- Archivo fuente
- Brecha detectada
- Accion requerida
- Estado

## Evidencias tipicas
- Politicas publicas y licencias.
- Descripcion de procesos de curacion.
- Declaraciones de versionado y preservacion.
- Guia de acceso a datos y APIs.
- Mecanismos de soporte y capacitacion.
